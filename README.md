# oceanadevs-api
#### by: Kevin Turado

##### Description: A work template built in typescript node and express. 

Install the libraries first by running the command:
`npm run install`

##### API's available: <br>
- authentication: 
    - login
    - logout
    - user session
- users:
    - create
    - get user by id
    - update user
    - get all user

##### Current Features:
  - Custom Error Handler
