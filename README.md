# oceanadevs-api
#### by: Kevin Turado

##### Description: A work template built in typescript node and express. 

Install the libraries first by running the command:
`npm run install`

##### API's available: <br>
    <ul>
      <li>authentication: </li>
       <ul>
        <li>login</li>
        <li>logout</li>
        <li>user session</li>
      </ul>
      <li>users:</li>
       <ul>
          <li>create</li>
          <li>get user by id</li>
          <li>update user</li>
          <li>get all user</li>
  </ul>

  ##### Current Features:
  - Custom Error Handler
